# NIP-09 Reference Card

### REFERENCE_CARD
NIP-09 Event Deletion Request: 
- Kind: 5
- Required tags: `e` or `a` (one or more), `k` (optional but recommended)
- Optional tags: `content` (text note)
- Filter patterns: `kind:5`, `tags:e:<event_id>`, `tags:a:<kind>:<pubkey>:<d-identifier>`
- Common use cases: 
    - Deleting accidental posts
    - Disowning events
    - Hiding or indicating deletion request status for referenced events
- Example JSON: 
```json
{
  "kind": 5,
  "pubkey": "<32-bytes hex-encoded public key of the event creator>",
  "tags": [
    ["e", "dcd59..464a2"],
    ["e", "968c5..ad7a4"],
    ["a", "<kind>:<pubkey>:<d-identifier>"],
    ["k", "1"],
    ["k", "30023"]
  ],
  "content": "these posts were published by accident"
}
```
