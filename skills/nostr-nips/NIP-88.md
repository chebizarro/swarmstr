# NIP-88 Reference Card

NIP-88 defines the event scheme for Polls on Nostr. 
- kind:1068: Poll event with content, option, relay, polltype, and endsAt tags.
- kind:1018: Response event with e tag referencing the poll event and response tags.
Required tags for kind:1068: content, option, relay.
Optional tags for kind:1068: polltype (default: singlechoice), endsAt.
Filter patterns: #e for referencing events, kinds for event types.
Example kind:1068 event: 
```json
{
  "content": "Pineapple on pizza",
  "kind": 1068,
  "tags": [
    ["option", "qj518h583", "Yay"],
    ["option", "gga6cdnqj", "Nay"],
    ["relay", "<relay url1>"],
    ["polltype", "singlechoice"],
    ["endsAt", "<unix timestamp in seconds>"]
  ]
}
```
