# NIP-27 Reference Card

NIP-27 standardizes inline references to events and profiles in readable text content. 
- Kind numbers: 1, 30023
- Required tags: None
- Optional tags: ["q", "<event-id> or <event-address>", "<relay-url>", "<pubkey-if-a-regular-event>"] for notifications and replies
- Filter patterns: `nostr:nprofile1...` or `nostr:npub1...` for profile mentions, `nostr:nevent1...` or `nostr:note1...` for event references
- Common use cases: 
    - Creating events with mentions to other profiles or events using NIP-21 codes
    - Notifying mentioned profiles or referenced events using NIP-18 quote tags
    - Displaying previews or links to mentioned profiles or events

Example:
```json
{
  "content": "hello nostr:nprofile1qqszclxx9f5haga8sfjjrulaxncvkfekj097t6f3pu65f86rvg49ehqj6f9dh",
  "tags": [
    [
      "p",
      "2c7cc62a697ea3a7826521f3fd34f0cb273693cbe5e9310f35449f43622a5cdc"
    ]
  ]
}
```
