# NIP-21 Reference Card

### REFERENCE_CARD
NIP-21 defines the `nostr:` URI scheme for maximum interoperability and openness in the network. The scheme is used with identifiers from NIP-19, except `nsec`. Examples include `nostr:npub...` for pubkeys, `nostr:nprofile...` for profiles, `nostr:note...` for notes, and `nostr:nevent...` for events. To link HTML pages to Nostr entities, use `<link>` tags with `rel="alternate"` for events and `rel="me"` or `rel="author"` for authorship. Common use cases include linking webpages to Nostr events or profiles, and assigning authorship of webpages to Nostr profiles.
