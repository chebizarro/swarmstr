# NIP-25 Reference Card

### REFERENCE_CARD
NIP-25 defines reactions as `kind 7` events for user reactions to other events. Required tags include `e` (event ID) and optional tags include `p` (pubkey), `a` (addressable event coordinates), and `k` (stringified kind number of reacted event). Filter patterns: `kind:7` for native Nostr reactions, `kind:17` for external content reactions. Common use cases: liking/disliking events, reacting with emojis. JSON examples:
```json
{
  "kind": 7,
  "content": "+",
  "tags": [
    ["e", "event_id", "relay_hint", "pubkey"],
    ["p", "pubkey", "relay_hint"]
  ]
}
```
