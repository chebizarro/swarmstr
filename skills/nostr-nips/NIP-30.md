# NIP-30 Reference Card

Custom emoji can be added to kind 0, 1, 7, and 30315 events using "emoji" tags in the form ["emoji", <shortcode>, <image-url>, <emoji-set-address>]. The shortcode must be alphanumeric and underscore-only. Image-url is required, while emoji-set-address is optional. Clients should parse shortcodes like :shortcode: to display custom emoji. Examples:
- Kind 0: {"kind": 0, "content": "{\"name\":\"Alex :soapbox:\"}", "tags": [["emoji", "soapbox", "https://example.com/soapbox.png"]]}
- Kind 1: {"kind": 1, "content": "Hello :gleasonator:", "tags": [["emoji", "gleasonator", "https://example.com/gleasonator.png"]]}
- Kind 7: {"kind": 7, "content": ":dezh:", "tags": [["emoji", "dezh", "https://example.com/dezh.svg"]]}
