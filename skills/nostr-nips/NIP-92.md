# NIP-92 Reference Card

NIP-92: Media Attachments. kind:1 (short text note) may include media URLs with matching `imeta` tags. `imeta` tags are variadic, with space-delimited key/value pairs, and MUST have a `url` and at least one other field. Example: `imeta` tag with `url`, `m` (mime type), `blurhash`, `dim` (dimensions), `alt` (alt text), `x` (sha256 hash), and `fallback` (alternative URL) fields. JSON example: `{ "content": "text with https://example.com/image.jpg", "kind": 1, "tags": [ [ "imeta", "url https://example.com/image.jpg", "m image/jpeg", ... ] ] }`. Filter pattern: `kind:1 AND tags:imeta`.
