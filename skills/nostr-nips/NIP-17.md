# NIP-17 Reference Card

### REFERENCE_CARD
NIP-17 defines an encrypted chat scheme using NIP-44 encryption and NIP-59 seals and gift wraps. 
- **Kind Numbers**: 
  - `14`: Chat message
  - `15`: File message
  - `1059`: Gift-wrapped event
  - `10050`: User's preferred relays for DMs
- **Required/Optional Tags**:
  - `p` (required for `kind 14` and `15`): Identifies one or more receivers of the message
  - `e` (optional): Denotes the direct parent message this post is replying to
  - `subject` (optional): Defines the current name/topic of the conversation
  - `relay` (required for `kind 10050`): Lists user's preferred relays for DMs
- **Filter Patterns**: 
  - `kind:14` for chat messages
  - `kind:15` for file messages
  - `kind:1059` for gift-wrapped events
- **Common Use Cases**:
  - Private direct messages between users
  - File sharing with encryption
  - Chat rooms with multiple participants
